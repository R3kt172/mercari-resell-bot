Mercari JP Resell Bot

Телеграм-бот для поиска выгодных позиций на японской барахолке Mercari. Заточен под реселл: помогает отлавливать интересные лоты, мониторить изменение цены и прикидывать потенциальную прибыль.

Что по функционалу

Бот стучится напрямую в API Mercari Japan через библиотеку mercapi. Это позволяет получать актуальные данные в обход ограничений веб-морды. Конвертация из йен в доллары идет по условному курсу 1 USD = 150 JPY.

Основные фишки:

    Поиск товаров по ключевым словам. Можно вбивать как английские названия, так и сразу на японском.

    Автоматический расчет профита, если указать предполагаемую розничную цену.

    Отслеживание конкретных позиций с сохранением истории цены.

    Топ товаров по уровню потенциальной прибыли.

    Ручная настройка минимального процента профита для фильтрации выдачи.

    Фотки товаров подгружаются прямо в чат.

Как поднять у себя

Сначала клонируем репозиторий и идем в папку проекта:
text

git clone https://github.com/yourusername/mercari-resell-bot.git
cd mercari-resell-bot

Дальше разворачиваем виртуальное окружение.

На Windows:
text

python -m venv venv
venv\Scripts\activate

На macOS или Linux:
text

python3 -m venv venv
source venv/bin/activate

Ставим зависимости:
text

pip install -r requirements.txt

Теперь нужно прописать переменные окружения. Создаем в корне проекта файл .env:
text

touch .env

Открываем его и добавляем строку с токеном бота:
text

BOT_TOKEN=ваш_токен_телеграм_бота

Токен получаем стандартно через BotFather в телеграме: команда /newbot, придумываем имя, забираем длинную строку с цифрами и буквами.

Все, запускаем:
text

python searchbot.py

Если в консоли появилось "Бот запущен и готов к работе!", значит все взлетело как надо.

Список команд
Команда	Что делает
/start	Приветствие и список доступных команд
/search запрос	Поиск товаров на Mercari
/check ссылка цена	Проверить конкретный лот и прикинуть прибыль
/track ссылка	Начать отслеживание позиции
/untrack ID	Снять товар с отслеживания
/top	Показать топ по прибыльности
/setmargin %	Задать минимальный процент профита для фильтрации
/mymargin	Глянуть, какой процент сейчас установлен

Пара примеров для понимания

Ищем кроссовки:
text

/search nike air max

Проверяем, выгодно ли брать конкретный лот, если планируем продать за 150 баксов:
text

/check https://mercari.jp/items/1234567890 15000

Ставим на мониторинг:
text

/track https://mercari.jp/items/1234567890

Что стоит держать в голове

Курс конвертации жестко зашит как 1 USD = 150 JPY. Это приблизительно, в реальности может плавать.

Mercari периодически режет частые запросы, так что не стоит спамить поиском без пауз.

Для работы отслеживания бот должен быть запущен непрерывно. Упал процесс — мониторинг встал.

Токен бота — вещь сугубо личная. Светить его в паблике или коммитить в репозиторий не надо.
